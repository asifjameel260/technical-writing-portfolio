# User Profile API Endpoint

This section describes the data structure returned when a developer requests a user's profile details.

## Sample JSON Response

```json
{
  "user_id": 4821,
  "display_name": "Priya Mehta",
  "account_status": "active"
}

Field Descriptions
user_id: A unique numeric identifier assigned to each user.
display_name: The full name of the user shown on their profile.
account_status: The current state of the account, such as active or suspended.
